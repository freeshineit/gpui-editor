# gpui editor

## 需求

- gpui(https://github.com/zed-industries/zed/tree/87cf32a6245f4a6b99805e28f0845d0e2e06b519/crates/gpui) 实现多文本编辑库(textarea)，并提供demo
- 支持多行编辑
- 支持像 html textarea 的功能
- 支持像 html textarea 的快捷键
- 支持像 html textarea 的光标, 自定义颜色，支持闪动
- 支持像 html textarea 选中背景和字体颜色
- 支持自定义背景和字体颜色
- 支持单元测试和集成测试
- 支持注释文档生成
- 支持展示和输入中文（多种语言）
- 支持展示和输入字体表情
- 支持事件回调和事件拦截回调

## 技术

- rust
- rust format
